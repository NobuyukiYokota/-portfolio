# Student Manager (C言語)

C言語で作成した学生管理プログラムです。
コンソール上で学生の情報を管理することができます。

## 機能

* 学生情報の登録
* 登録データの表示
* 平均点の計算
* データのファイル保存

## メニュー

```
1. 登録
2. 表示
3. 平均
4. 保存
5. 終了
```

## 使用技術

* C言語
* 配列
* 構造体
* ファイル入出力

## ファイル

* student_manager.c
  メインプログラム
